# SKILLs

## ユーザ呼び出し

ユーザからのスキル呼び出しでアクセス可能（`disable-model-invocation: true`）。

- [grill-with-docs](./grill-with-docs/SKILL.md) - プロジェクトのドメインモデルを構築し、用語を明確にし、`CONTEXT.md`とADRをインラインで更新する、グリルセッション。
- [grill-me](./grill-me/SKILL.md) - 計画や設計について、意思決定ツリーのすべての分岐が解決されるまで、インタビューを行う。
- [improve-codebase-architecture](./improve-codebase-architecture/SKILL.md) - コードベースをスキャンして改善の機会を探し出し、それらを視覚的なHTMLレポートとして提示し、その後、選択したレポートを徹底的に検証する。

## モデルが呼び出し

モデルからの自発的な呼び出し

- [diagnosing-bugs](./diagnosing-bugs/SKILL.md) - 深刻なバグやパフォーマンス低下に対する規律ある診断ループ：再現 → 最小化 → 仮説を立てる → 計測 → 修正 → リグレッションテスト。
- [domain-modeling](./domain-modeling/SKILL.md) - プロジェクトのドメインモデルを積極的に構築し、磨き上げます。用語を検証し、シナリオを使用してストレステストを行い、`CONTEXT.md`とADRをインラインで更新します。
- [codebase-design](./codebase-design/SKILL.md) - ディープモジュールを設計するための共通の規律と語彙：小さなインターフェース、明確な継ぎ目、インターフェースを介したテスト可能性。
- [grilling](./grilling/SKILL.md) -  計画やデザインについて、意思決定ツリーのすべての分岐が解決されるまで、ユーザーに徹底的にインタビューを行う。
