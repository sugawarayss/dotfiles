## setup
Run this:
```bash
# update ~/.zshrc & ~/.zprofile
make zsh
```

```bash
# set commit_template for git config global
make git
```

```bash
# set up nvim 
make vim
```

```bash
# brew bundle by Brewfile
make homebrew
```

```bash
# workflows & keybinding
make warp
```

```bash
# setup zsh, git vim homebrew warp
make all
```
